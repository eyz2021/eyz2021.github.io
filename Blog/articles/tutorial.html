Python 自动化办公：Excel 数据处理极简示例

2025-09-06 • 8分钟阅读 • 编程

日常办公中，Excel 数据筛选和统计是常见需求。本文用 20行Python代码 实现核心功能：读取Excel、筛选有效数据、按地区统计销量，帮你快速上手自动化办公。

工具：Python + pandas（轻量高效的数据处理库）

需求：从「销售数据.xlsx」中筛选销量>0的记录，按地区统计总销量。

一、1分钟准备：安装依赖库

只需安装 pandas（自带Excel读写功能），执行以下命令：

# 安装pandas（含Excel处理模块）
pip install pandas openpyxl

二、核心代码：20行实现Excel自动化

以下代码完成「读取→筛选→统计→保存」全流程，注释清晰，可直接复用：

import pandas as pd

# 1. 读取Excel文件（1行）
df = pd.read_excel("销售数据.xlsx", engine="openpyxl")

# 2. 筛选有效数据：删除空行 + 销量>0（2行）
df_clean = df.dropna()  # 删除空行
df_clean = df_clean[df_clean["销量"] > 0]  # 筛选销量>0的记录

# 3. 按地区统计总销量（2行）
area_stats = df_clean.groupby("地区")["销量"].sum().reset_index()
area_stats.columns = ["地区", "总销量"]  # 重命名列名

# 4. 保存结果到新Excel（2行）
with pd.ExcelWriter("销售数据_处理后.xlsx", engine="openpyxl") as writer:
    df_clean.to_excel(writer, sheet_name="清洗后数据", index=False)
    area_stats.to_excel(writer, sheet_name="地区销量统计", index=False)

# 5. 打印结果验证（2行）
print("清洗后数据行数：", len(df_clean))
print("\n按地区统计结果：")
print(area_stats)

三、关键说明与扩展

1. 数据格式要求

Excel文件需包含至少两列：地区（文本类型）、销量（数字类型），示例数据如下：

地区 产品 销量
北京 手机 120
上海 电脑 80

2. 扩展功能推荐

如需增加功能，可在核心代码基础上补充：

  • 日期格式统一：df["日期"] = pd.to_datetime(df["日期"]).dt.strftime("%Y-%m-%d")
  • 销售额计算：df["销售额"] = df["销量"] * df["单价"]
  • 数据导出为CSV：df_clean.to_csv("清洗后数据.csv", index=False, encoding="utf-8-sig")
